Habit (습관) + Tree (나무) = Habitree

브랜드 철학

Habitree는 보이지 않는 습관을 보이는 자산으로 전환하는 플랫폼 생태계입니다.

Mission

보이지 않는 습관을 보이는 자산으로

개발 설정, 독서 기록 등 매일 반복하지만 쉽게 사라지는 습관들을 체계적으로 축적하여 개인의 지적 자산으로 전환합니다.

Core Insight

잃어버리지 않으면, 자산이 됩니다

흩어진 API 키, 잊혀진 밑줄. 사라지지 않게 기록하면, 그것이 곧 당신의 자산이 됩니다.

Problem: Developer

Stripe 키를 잃어버렸다

Notion, 메모장, .env 파일 사이에 흩어진 API 키와 설정. 중요한 연결이 단절되는 순간, 개발 속도가 멈춥니다.

Problem: Reader

갤러리 속에 묻힌 책 사진

밑줄 치고 찍은 사진은 갤러리에 묻히고, 감동받은 문장은 기억 속에서 사라집니다. 독서의 흔적이 자산으로 남지 않습니다.

연결 포인트

Linkmap → ReadTree

Linkmap의 서비스 카탈로그에 ReadTree를 등록. 개발자가 독서 기록 API를 자신의 프로젝트에 연결할 수 있는 템플릿 제공.

ReadTree → Linkmap

ReadTree 사용자가 기록한 개발 서적의 인사이트를 Linkmap의 프로젝트 설정에 연결. 읽은 기술 서적 기반 스택 추천.

Habitree 통합 계정

하나의 Habitree 계정으로 Linkmap과 ReadTree를 모두 사용. 통합 대시보드에서 개발 습관과 독서 습관을 한눈에 확인하고, 두 영역의 성장 궤적을 추적합니다.
