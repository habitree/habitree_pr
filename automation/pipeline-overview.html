Linkmap

n8n Video Pipeline

n8n 워크플로우 기반 풀스택 영상 자동화. Gemini AI 스크립트 생성부터 YouTube 배포까지.

Google Sheets 큐에서 대기 콘텐츠 조회
소스 수집 (텍스트/URL/GDoc/Sheet)
Gemini AI 스크립트 생성 (팟캐스트/내레이션/쇼츠)
JSON2Video / Creatomate 영상 렌더링
Google Drive 업로드 및 저장
YouTube SEO 메타데이터 생성 + 업로드
ReadTree

Remotion Shorts Pipeline

React + Remotion 기반 숏폼 영상 자동 생성. Claude API 스크립트 + ElevenLabs TTS.

Supabase DB에서 콘텐츠 데이터 입력
Claude API로 한국어 내레이션 스크립트 생성
ElevenLabs TTS 음성 생성
Remotion React 컴포넌트로 영상 렌더링
음성 + 영상 + BGM + 자막 합성 (MP4)
QA 검증 후 Instagram/YouTube/TikTok 업로드
