2026년 현재, 미국 개발자의 92%가 AI 코딩 도구를 매일 사용하고 전 세계 코드의 41%가 AI로 생성됩니다. Andrej Karpathy가 명명한 “바이브 코딩” 패러다임은 코드 생성의 진입 장벽을 극적으로 낮췄습니다. As of 2026, 92% of US developers use AI coding tools daily and 41% of all code worldwide is AI-generated. The “vibe coding” paradigm coined by Andrej Karpathy has dramatically lowered the barrier to code generation. 截至2026年，美国92%的开发者每天使用AI编码工具，全球41%的代码由AI生成。Andrej Karpathy提出的“Vibe编程”范式极大地降低了代码生成的门槛。

  • “GitHub에 올렸더니 보안 경고가 떴어요”“I pushed to GitHub and got a security alert”“我推送到GitHub后收到了安全警告”

React Flow 기반 인터랙티브 다이어그램으로, 프로젝트에 연결된 모든 서비스(GitHub, Supabase, Stripe, Cloudflare 등)의 관계를 한 화면에 시각화합니다. 연결 유형(API/웹훅/SDK)과 상태를 직관적으로 파악할 수 있어 팀원 온보딩 시간이 대폭 줄어듭니다. An interactive diagram built on React Flow that visualizes the relationships of all connected services (GitHub, Supabase, Stripe, Cloudflare, etc.) on one screen. Connection types (API/webhook/SDK) and status are intuitively displayed, drastically reducing team onboarding time. 基于React Flow构建的交互式图表，将项目连接的所有服务（GitHub、Supabase、Stripe、Cloudflare等）的关系在一个屏幕上可视化。直观显示连接类型（API/Webhook/SDK）和状态，大幅缩短团队入职时间。

원클릭 배포One-Click Deploy一键部署

템플릿 선택 → GitHub 연결 → 배포 버튼 — 이 세 단계만으로 약 3분 안에 개인 홈페이지가 GitHub Pages에 올라갑니다. Git 명령어를 모르는 완전 초보자도 부담 없이 첫 배포를 경험하고, 이후 같은 계정에서 서비스 맵, 환경변수, 팀 협업까지 확장할 수 있습니다. Pick a template → Connect GitHub → Click Deploy — in just three steps, your personal website goes live on GitHub Pages in about 3 minutes. Even complete beginners who don't know Git commands can experience their first deployment, then expand to service maps, env variables, and team collaboration on the same account. 选择模板 → 连接GitHub → 点击部署按钮——仅需三步，约3分钟内您的个人网站就会发布到GitHub Pages上。即使完全不懂Git命令的新手也能轻松完成首次部署，之后可在同一账户中扩展到服务地图、环境变量和团队协作。

AES-256-GCM 암호화 환경변수AES-256-GCM Encrypted Environment VariablesAES-256-GCM加密环境变量

API 키와 시크릿을 AES-256-GCM으로 서버 사이드 암호화하여 저장합니다. 개발/스테이징/프로덕션 환경별 분리, .env 원클릭 다운로드, GitHub Secrets 자동 동기화를 지원하며, 모든 접근·변경은 감사 로그에 기록됩니다. 복호화된 값은 절대 로깅되지 않습니다. API keys and secrets are stored with AES-256-GCM server-side encryption. Supports dev/staging/production environment separation, one-click .env download, and GitHub Secrets auto-sync. All access and changes are recorded in audit logs. Decrypted values are never logged. API密钥和密钥信息通过AES-256-GCM服务端加密存储。支持开发/预发/生产环境分离、一键下载.env文件和GitHub Secrets自动同步。所有访问和变更都记录在审计日志中。解密后的值绝不会被记录。

GitHub 연동은 어떻게 되나요?How does GitHub integration work?GitHub集成是如何运作的？
GitHub OAuth 로그인 후, GitHub Secrets 자동 동기화 기능으로 Linkmap에 등록한 환경변수를 GitHub Actions에서 바로 사용할 수 있습니다. NaCl 암호화로 전송하며, 원클릭 배포 시 자동으로 레포지토리 생성·코드 푸시·GitHub Pages 배포까지 완료됩니다.
MCP Server가 뭔가요?What is the MCP Server?什么是MCP服务器？
Model Context Protocol(MCP) Server는 Claude Code, Cursor 같은 AI 코딩 도구에서 직접 Linkmap API를 호출할 수 있게 해주는 인터페이스입니다. AI가 코드를 작성하면서 동시에 환경변수를 조회하고, 서비스를 등록하고, 프로젝트 설정을 관리할 수 있습니다 — 터미널이나 브라우저를 오갈 필요 없이.
기존 프로젝트에도 적용할 수 있나요?Can I use it with existing projects?可以用于现有项目吗？
네. 새 프로젝트를 만들어 기존 서비스를 하나씩 등록하면 됩니다. CLI 도구(linkmap pull)로 기존 .env 파일을 일괄 임포트할 수 있으며, 서비스 연결은 카탈로그에서 선택하여 추가합니다. 이미 운영 중인 서비스도 서비스 맵에 등록하면 전체 아키텍처를 시각화할 수 있습니다.
데이터는 어디에 저장되나요?Where is data stored?数据存储在哪里？
Supabase(AWS 기반 PostgreSQL)에 저장되며, 환경변수는 AES-256-GCM으로 암호화된 상태로만 저장됩니다. 애플리케이션은 Cloudflare Workers 글로벌 엣지에 배포되어 한국을 포함한 전 세계에서 빠르게 접근할 수 있습니다. Row Level Security(RLS)로 사용자 간 데이터 격리를 보장합니다.
